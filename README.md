# FARM Stack Monorepo Application
This a learning project from this I am learning the FARM Stack .
This is a full-stack web application built using the FARM stack (FastAPI, React, MongoDB).  
The project demonstrates end-to-end integration between a Python-based backend and a modern React frontend.

It includes RESTful APIs, database operations, and a responsive UI to manage data efficiently.

## Features

- User authentication (Login / Signup)
- RESTful API using FastAPI
- CRUD operations (Create, Read, Update, Delete)
- MongoDB database integration
- React frontend with API integration
- Responsive UI

## Tech Stack

### Backend
- FastAPI
- Python
- MongoDB

### Frontend
- React
- Axios / Fetch API

### Tools
- Git & GitHub
- Postman
